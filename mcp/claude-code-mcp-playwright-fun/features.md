# UI Features

- Page title "Product Manager"
- "Add New Product" button to show/hide the product form
- Product form with fields: Name, Price, Date, URL
- "Save Product" button to submit new product
- "Cancel" button to hide form without saving
- Products table with columns: Name, Price, Created At, Action
- Alternating row colors in table (white/gray)
- Click on any table row to open product URL in new tab
- "View" button on each row to open product URL
- Pre-loaded products: iPhone 15, MacBook Pro, Apple Watch
