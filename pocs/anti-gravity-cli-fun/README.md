# Anti-Gravity Cli

Google is killing Gemini cli and migrating to anti-gravity-cli.

## Specifications
- **Model**: Gemini 3.5 Flash (High)
- **Antigravity CLI Version**: 1.0.0

## Application
- [app-with-anti-gravity-cli](app-with-anti-gravity-cli)

## Install

```bash
curl -fsSL https://antigravity.google/cli/install.sh | bash
```

```
$HOME/.local/bin/agy
```

### Run 

```
agy
```

### Experience notes

```
+ Easy to install
+ fast
+ Code result was decent
```
