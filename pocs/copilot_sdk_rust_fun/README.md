
### Run

```bash
cargo run
```

### Requirements

- Rust 1.85+ (Edition 2024)
- GitHub Copilot CLI installed and authenticated
- `copilot` in `PATH` or set `COPILOT_CLI_PATH`

### Usage

Type a city name and press Enter.
Type `exit` or `quit` to stop.
