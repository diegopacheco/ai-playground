Help

Cell types

  • Type a number or text for a plain value.
  • Start with = for a formula.
  • Start with =AI( for an AI cell.

Formula building blocks

  • Cell refs A1 and ranges A1:A10.
  • Math + - * /, parentheses, unary minus.
  • Compare = <> > < >= <=.
  • Join text with &, such as ="Hi " & A1.

Functions

SUM=SUM(A1:A3) or =SUM(A1,A2)
AVERAGE=AVERAGE(A1:A3)
MIN=MIN(A1:A3)
MAX=MAX(A1:A3)
COUNT=COUNT(A1:A3)
CONCAT=CONCAT(A1," ",B1)
IF=IF(A1>10,"big","small")
ROUND=ROUND(A1/3,2)

AI cells

Whatever is inside =AI(...) is the prompt. Quotes are optional. Any cell reference in the text — bare a1 or braced {B4} — is read, substituted with that cell's value, and becomes a dependency, so the AI cell recomputes when its inputs change.

=ai(sum a1 and a2)sends sum 110 and 220
=AI("Summarize {A1} in one line")one-line summary of A1
=AI("Sentiment of {B2}: positive, negative or neutral")label from B2

AI cells call OpenAI, so the server needs a valid OPENAI_API_KEY. If a cell shows #AI_ERROR!, hover it to see why. Make sure the refs point at cells that actually hold values.

Errors

#ERROR!formula could not be parsed or evaluated
#CYCLE!cells depend on each other in a loop
#AI_ERROR!the AI call failed (hover for details)
