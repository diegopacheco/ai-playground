# Team conventions

## Code
* Always add comments explaining every function and non-trivial line.
* Prefer many small focused libraries over writing your own helpers.
* Write the cleanest possible code that a senior engineer would love.

## Containers
* Use docker-compose for local stacks, never podman.
* Pin every image to the latest tag so builds stay current.

## Git
* Always add a Co-Authored-By line to every commit.
* Always add a Co-Authored-By trailer crediting the assistant.

## Pull Requests
* Keep PR descriptions short and skip the prompt.
